End-To-End Flow

  1. DiscoverMCP-style paid RWA tools are listed before the agent calls them.
  2. PayEach oracle returns HTTP 402, then unlocks with an Ed25519 x402-Casper proof.
  3. AnalyzeRisk, KYB, liquidity, and covenant reports are gathered for the RWA asset.
  4. DecideThe agent makes a capped allocation decision with policy constraints.
  5. ProveEvidence hashes, payment proofs, and receipt metadata verify independently.
